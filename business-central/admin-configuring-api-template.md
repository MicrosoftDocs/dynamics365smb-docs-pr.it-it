<span data-ttu-id="ee04f-117">Se è presente più di un modello selezionato per un'API (ID pagina), i modelli vengono applicati nell'ordine definito nella colonna **Order**.</span><span class="sxs-lookup"><span data-stu-id="ee04f-117">If there is more than one template selected for an API (Page ID), the templates are applied in the order defined in the **Order** column.</span></span>

3. <span data-ttu-id="ee04f-119">Selezionare un valore **ID pagina**.</span><span class="sxs-lookup"><span data-stu-id="ee04f-119">Select a **Page ID** value.</span></span>  
<span data-ttu-id="ee04f-120">Questa è la pagina per l'API a cui verrà applicato il modello.</span><span class="sxs-lookup"><span data-stu-id="ee04f-120">This is the page for the API to which the template will be applied.</span></span> <span data-ttu-id="ee04f-121">La ricerca **ID pagina** fornisce un elenco di tutte le API disponibili nella libreria.</span><span class="sxs-lookup"><span data-stu-id="ee04f-121">The **Page ID** lookup provides a list of all APIs available in the library.</span></span>
4. <span data-ttu-id="ee04f-122">Selezionare un valore nel campo **Codice modello**.</span><span class="sxs-lookup"><span data-stu-id="ee04f-122">Select a value in the **Template Code** field.</span></span>  
<span data-ttu-id="ee04f-123">Il codice modello è il codice per il modello definito nella pagina **Modelli di configurazione**.</span><span class="sxs-lookup"><span data-stu-id="ee04f-123">The template code is the code for the template that was defined on the **Configuration Templates** page.</span></span> <span data-ttu-id="ee04f-124">I valori del modello definiti vengono applicati all'API.</span><span class="sxs-lookup"><span data-stu-id="ee04f-124">The template values defined are applied to the API.</span></span> 
5. <span data-ttu-id="ee04f-125">Nel campo **Condizioni** specificare quale modello deve essere applicato.</span><span class="sxs-lookup"><span data-stu-id="ee04f-125">In the **Conditions** field, specify which template should be applied.</span></span>  
<span data-ttu-id="ee04f-126">Il modello definito viene applicato a un nuovo record creato tramite l'API se, e solo se, le condizioni definite nel campo **Condizioni** sono soddisfatte dai valori già definiti per la nuova istanza dell'entità.</span><span class="sxs-lookup"><span data-stu-id="ee04f-126">The defined template is applied to a new record created through the API if, and only if, the conditions defined in the **Conditions** field are met by the values already defined for the new instance of the entity.</span></span>
