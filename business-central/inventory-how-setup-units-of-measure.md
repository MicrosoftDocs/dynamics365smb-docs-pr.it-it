# <a name="set-up-item-units-of-measure"></a><span data-ttu-id="b657d-103">Impostare unità di misura articolo</span><span class="sxs-lookup"><span data-stu-id="b657d-103">Set Up Item Units of Measure</span></span>
<span data-ttu-id="b657d-104">È possibile impostare più unità di misura per un articolo in modo da poter assegnare le unità di misura all'articolo per i seguenti scopi:</span><span class="sxs-lookup"><span data-stu-id="b657d-104">You can set up multiple units of measure for an item so that you can assign units of measure to the item for the following purposes:</span></span>

- <span data-ttu-id="b657d-105">Assegnare un'unità di misura base sulla scheda articolo per definire come viene memorizzata nel magazzino e per essere utilizzata come base di conversione per le unità di misura alternative.</span><span class="sxs-lookup"><span data-stu-id="b657d-105">Assign a base unit of measure on the item’s item card to define how it is stored in inventory and to serve as the conversion basis for alternate units of measure.</span></span>
- <span data-ttu-id="b657d-106">Assegnare unità di misura alternative ai documenti di acquisto, produzione, o vendita per indicare quante unità dell'unità di misura di base alla volta gestire nei processi.</span><span class="sxs-lookup"><span data-stu-id="b657d-106">Assign alternate units of measure to purchase, production, or sales documents to specify how many units of the base unit of measure you handle at a time in those processes.</span></span> <span data-ttu-id="b657d-107">Ad esempio, è possibile acquistare l'articolo in pallet e utilizzare solo i pezzi singoli nella produzione.</span><span class="sxs-lookup"><span data-stu-id="b657d-107">For example, you may buy the item on pallets and only use single pieces in your production.</span></span>

<span data-ttu-id="b657d-108">Se un articolo viene inserito in stock utilizzando un'unità di misura ma viene prodotto in un'altra, viene creato un ordine di produzione che utilizza un'unità di misura batch di produzione per calcolare la quantità corretta di componenti durante il processo batch **Aggiorna ordine produzione**.</span><span class="sxs-lookup"><span data-stu-id="b657d-108">If an item is stocked in one unit of measure but produced in another, a production order is created that uses a manufacturing batch unit of measure to calculate the correct quantity of the components during the **Refresh Production Order** batch job.</span></span> <span data-ttu-id="b657d-109">Un esempio di calcolo di un'unità di misura è il caso in cui un articolo prodotto viene inserito in stock in pezzi, ma viene prodotto in tonnellate.</span><span class="sxs-lookup"><span data-stu-id="b657d-109">An example of a manufacturing batch unit of measure calculation is when a manufactured item is stocked in pieces but produced in tons.</span></span> <span data-ttu-id="b657d-110">Per ulteriori informazioni, vedere [Utilizzare le unità di misura batch di produzione](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span><span class="sxs-lookup"><span data-stu-id="b657d-110">For more information, see [Work with Manufacturing Batch Units of Measure](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span></span>

## <a name="to-set-up-a-unit-of-measure"></a><span data-ttu-id="b657d-111">Per impostare un'unità di misura</span><span class="sxs-lookup"><span data-stu-id="b657d-111">To set up a unit of measure</span></span>
1. <span data-ttu-id="b657d-112">Scegliere l'icona a forma di ![lampadina che consente di aprire la funzionalità delle informazioni](media/ui-search/search_small.png "Informazioni sull'operazione che si desidera eseguire"), immettere **Articoli** e quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="b657d-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="b657d-113">Aprire la scheda dell'articolo per la quale si intende impostare unità di misura alternative.</span><span class="sxs-lookup"><span data-stu-id="b657d-113">Open the card of the item for which you want to set up alternate units of measure.</span></span>
3. <span data-ttu-id="b657d-114">Scegliere l'azione **Unità di misura**.</span><span class="sxs-lookup"><span data-stu-id="b657d-114">Choose the **Units of Measure** action.</span></span> <span data-ttu-id="b657d-115">Verrà visualizzata la pagina **Unità di misura articoli**.</span><span class="sxs-lookup"><span data-stu-id="b657d-115">The **Item Units of Measure** page opens.</span></span>
4. <span data-ttu-id="b657d-116">Se il campo **Unità di misura base** nella scheda articolo è compilato, allora tale unità di misura è già impostata.</span><span class="sxs-lookup"><span data-stu-id="b657d-116">If the **Base Unit of Measure** field on the item card is filled, then that unit of measure is already set up.</span></span>
5. <span data-ttu-id="b657d-117">Scegliere l'azione **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="b657d-117">Choose the **New** action.</span></span> <span data-ttu-id="b657d-118">Una nuova riga vuota viene inserita.</span><span class="sxs-lookup"><span data-stu-id="b657d-118">A new empty line is inserted.</span></span>
6. <span data-ttu-id="b657d-119">Nel campo **Codice** immettere il nome dell'unità di misura.</span><span class="sxs-lookup"><span data-stu-id="b657d-119">In the **Code** field, enter the name of the unit of measure.</span></span> <span data-ttu-id="b657d-120">In alternativa, selezionare il campo per selezionare i codici di unità di misura che sono nel database.</span><span class="sxs-lookup"><span data-stu-id="b657d-120">Alternatively, choose the field to select from the unit of measure codes that are in the database.</span></span>
7. <span data-ttu-id="b657d-121">Nel campo **Qtà per unità di misura** immettere il numero dell'unità di misura di base che la nuova unità di misura contiene.</span><span class="sxs-lookup"><span data-stu-id="b657d-121">In the **Qty. per Unit of Measure** field, enter how many units of the base unit of measure the new unit of measure contains.</span></span>
8. <span data-ttu-id="b657d-122">Ripetere i passaggi da 5 a 7 per impostare tutte le unità di misura alternative da utilizzare nei processi diversi per questo articolo.</span><span class="sxs-lookup"><span data-stu-id="b657d-122">Repeat steps 5 through 7 to set up all the alternate units of measure that you want to use in different processes for this item.</span></span>

<span data-ttu-id="b657d-123">È ora possibile utilizzare le unità di misura alternative nei documenti di acquisto, produzione e vendita.</span><span class="sxs-lookup"><span data-stu-id="b657d-123">You can now use the alternate units of measure on purchase, production, and sales documents.</span></span> <span data-ttu-id="b657d-124">Per ulteriori informazioni, vedere Immettere codici di unità di misura di default per transazioni di acquisto e di vendita o Utilizzare l'unità di misura batch di produzione.</span><span class="sxs-lookup"><span data-stu-id="b657d-124">For more information, see Enter Default Units of Measure Codes for Purchase Transactions and Sales Transactions or Use the Manufacturing Batch Unit of Measure.</span></span>

## <a name="to-set-up-unit-of-measure-translations"></a><span data-ttu-id="b657d-125">Per impostare traduzioni di unità di misura</span><span class="sxs-lookup"><span data-stu-id="b657d-125">To set up unit of measure translations</span></span>
<span data-ttu-id="b657d-126">Se si vendono articoli a clienti stranieri, potrebbe essere necessario indicare l'unità di misura nella lingua del cliente.</span><span class="sxs-lookup"><span data-stu-id="b657d-126">When you sell items to foreign customers, you may want to specify the unit of measure in the customer’s language.</span></span> <span data-ttu-id="b657d-127">Questa operazione può essere eseguita impostando le necessarie traduzioni delle unità di misura.</span><span class="sxs-lookup"><span data-stu-id="b657d-127">You can do this after you have set up the necessary unit of measure translations.</span></span>

1. <span data-ttu-id="b657d-128">Scegliere l'icona a forma di ![lampadina che consente di aprire la funzionalità delle informazioni](media/ui-search/search_small.png "Informazioni sull'operazione che si desidera eseguire"), immettere **Unità di misura** e quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="b657d-128">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Units of Measure**, and then choose the related link.</span></span>
2. <span data-ttu-id="b657d-129">Selezionare il codice per cui si intendono impostare le traduzioni e scegliere l'azione **Traduzioni**.</span><span class="sxs-lookup"><span data-stu-id="b657d-129">Select the code for which you want to set up translations, and then choose the **Translations** action.</span></span>
3. <span data-ttu-id="b657d-130">Nel campo **Cod. lingua** selezionare la freccia a discesa per visualizzare una lista dei codici lingua disponibili.</span><span class="sxs-lookup"><span data-stu-id="b657d-130">In the **Language Code** field, select the drop-down arrow to see a list of available language codes.</span></span> <span data-ttu-id="b657d-131">Selezionare il codice lingua per il quale si intende immettere la traduzione, quindi fare clic su OK per copiare il codice nel campo.</span><span class="sxs-lookup"><span data-stu-id="b657d-131">Select the language code for which you want to enter a translation, and then choose the OK button to copy the code to the field.</span></span>
4. <span data-ttu-id="b657d-132">Nel campo **Descrizione**, immettere il testo adeguato.</span><span class="sxs-lookup"><span data-stu-id="b657d-132">In the **Description** field, enter the appropriate text.</span></span>
5. <span data-ttu-id="b657d-133">Ripetere i passaggi da 2 a 4 per i codici delle unità di misura e per le lingue per le quali si intendono inserire le traduzioni.</span><span class="sxs-lookup"><span data-stu-id="b657d-133">Repeat steps 2 through 4 for the unit of measure codes and the languages for which you want to enter translations.</span></span>

1. <span data-ttu-id="b657d-137">Scegliere l'icona a forma di ![lampadina che consente di aprire la funzionalità delle informazioni](media/ui-search/search_small.png "Informazioni sull'operazione che si desidera eseguire"), immettere **Articoli** e quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="b657d-137">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="b657d-138">Aprire la scheda articolo pertinente per la quale si intende specificare un codice di unità di misura di vendita o acquisto di default.</span><span class="sxs-lookup"><span data-stu-id="b657d-138">Open the relevant item card for which you want to specify a default sales or purchase unit of measure code.</span></span>
3. <span data-ttu-id="b657d-139">Per le vendite, nella Scheda dettaglio **Fatturazione**, nel campo **Unità di misura vendita** , aprire la pagina **Unità di misura articoli** .</span><span class="sxs-lookup"><span data-stu-id="b657d-139">For sales, on the **Invoicing** FastTab, in the **Sales Unit of Measure** field, open the **Item Units of Measure** page.</span></span>

Unit of Measure** field, open the **Item Units of Measure** page.</span></span>
5. <span data-ttu-id="b657d-141">Selezionare il codice che si intende impostare come unità di misura di default per le vendite o gli acquisti rispettivamente quindi scegliere il pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="b657d-141">Select the code you want to set up as the default unit of measure for sales or purchasing respectively, and then choose the **OK** button.</span></span>
