## <a name="to-print-quarterly-or-monthly-reports"></a><span data-ttu-id="b8190-110">Per stampare report trimestrali o mensili</span><span class="sxs-lookup"><span data-stu-id="b8190-110">To print quarterly or monthly reports</span></span>  

1.  <span data-ttu-id="b8190-111">Scegliere l'icona ![Cerca pagina o report](../../media/ui-search/search_small.png "Cerca pagina o report"), immettere **Registrazioni Intrastat**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="b8190-111">Choose the ![Search for Page or Report](../../media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Intrastat Journals**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="b8190-112">Selezionare il batch contabile desiderato.</span><span class="sxs-lookup"><span data-stu-id="b8190-112">Select the required journal batch.</span></span>  
3.  <span data-ttu-id="b8190-113">Per aprire il report **Intrastat - Report mensile** o **Intrastat - Report trimestrale**, scegliere l'azione **Report mensile** o **Report trimestrale**.</span><span class="sxs-lookup"><span data-stu-id="b8190-113">To open the **Intrastat - Monthly** report or the **Intrastat - Quarterly** report, choose the **Monthly Report** action or the **Quarterly Report** action.</span></span>

6.  <span data-ttu-id="b8190-116">Nella finestra **Registrazioni Intrastat**, per salvare i dettagli del report in un file, nella scheda **Azioni** nel gruppo, **Stampa**, scegliere **Crea file**.</span><span class="sxs-lookup"><span data-stu-id="b8190-116">In the **Intrastat Journal** window, to save the report details to a diskette, on the **Actions** tab, in the **Print** group, choose **Make Diskette**.</span></span>

    >  <span data-ttu-id="b8190-117">Prima della stampa, è possibile selezionare il percorso e il nome file.</span><span class="sxs-lookup"><span data-stu-id="b8190-117">Before you print, you can select the path and file name.</span></span> <span data-ttu-id="b8190-118">Se non vengono specificate queste informazioni, il file verrà denominato **scambi.cee** e verrà stampato nella radice dell'unità C.</span><span class="sxs-lookup"><span data-stu-id="b8190-118">If you do not specify this information, the file will be named **scambi.cee** and will print to the root of the C drive.</span></span>
