# <a name="import-and-export-a-report-or-document-layout"></a><span data-ttu-id="a24da-103">Importare ed esportare un layout di report o documento</span><span class="sxs-lookup"><span data-stu-id="a24da-103">Import and Export a Report or Document Layout</span></span>
<span data-ttu-id="a24da-104">È possibile esportare e importare un layout di report personalizzato esistente come file da e verso un computer o una rete.</span><span class="sxs-lookup"><span data-stu-id="a24da-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="a24da-105">Ad esempio, è possibile esportare un layout di report e inviare il file a un'altra persona per essere modificato.</span><span class="sxs-lookup"><span data-stu-id="a24da-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="a24da-106">Dopo le modifiche, il file può essere importato nuovamente.</span><span class="sxs-lookup"><span data-stu-id="a24da-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  
  
> [!IMPORTANT]  
>  <span data-ttu-id="a24da-107">Non è possibile importare o esportare i layout di report predefiniti.</span><span class="sxs-lookup"><span data-stu-id="a24da-107">You cannot import or export built-in report layouts.</span></span>  
  
### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="a24da-108">Per esportare un layout di report verso un file</span><span class="sxs-lookup"><span data-stu-id="a24da-108">To export a report layout to a file</span></span>  
  
1.  <span data-ttu-id="a24da-109">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "icona Cerca pagina o report"), immettere **Selezione layout report**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a24da-109">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
2.  <span data-ttu-id="a24da-110">Selezionare la riga del report che contiene il layout di report personalizzato che si desidera esportare, quindi nella scheda **Pagina iniziale**, nel gruppo **Processo** scegliere **Personalizza layout**.</span><span class="sxs-lookup"><span data-stu-id="a24da-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
3.  <span data-ttu-id="a24da-111">Nella finestra **Layout report**, selezionare il layout di report che si desidera esportare verso un file, quindi nel gruppo **Processo** della scheda **Pagina iniziale**, scegliere **Esporta layout**.</span><span class="sxs-lookup"><span data-stu-id="a24da-111">In the **Report Layouts** window, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  
  
4.  <span data-ttu-id="a24da-112">Nella finestra di dialogo **Esporta file**, selezionare **Salva** quindi salvare il campo in un percorso del computer o in rete.</span><span class="sxs-lookup"><span data-stu-id="a24da-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  
  
### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="a24da-113">Per importare un file di layout di report</span><span class="sxs-lookup"><span data-stu-id="a24da-113">To import a report layout file</span></span>  
  
1.  <span data-ttu-id="a24da-114">Assicurarsi che il file pertinente che definisce il layout di report sia disponibile nel computer o in rete.</span><span class="sxs-lookup"><span data-stu-id="a24da-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  
  
     <span data-ttu-id="a24da-115">Un file del layout di report Word deve avere l'estensione .docx.</span><span class="sxs-lookup"><span data-stu-id="a24da-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="a24da-116">Un file del layout di report RDLC deve avere l'estensione .rdl o .rdlc.</span><span class="sxs-lookup"><span data-stu-id="a24da-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  
  
2.  <span data-ttu-id="a24da-117">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "icona Cerca pagina o report"), immettere **Selezione layout report**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="a24da-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
3.  <span data-ttu-id="a24da-118">Selezionare la riga del report verso cui si desidera importare il layout di report, quindi nella scheda **Pagina iniziale**, nel gruppo **Processo** scegliere **Personalizza layout**.</span><span class="sxs-lookup"><span data-stu-id="a24da-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
4.  <span data-ttu-id="a24da-119">Nella finestra **Layout report**, selezionare il layout di report in cui importare il file, quindi nel gruppo **Processo** della scheda **Pagina iniziale**, scegliere **Importa layout**.</span><span class="sxs-lookup"><span data-stu-id="a24da-119">In the **Report Layouts** window, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  
  
5.  <span data-ttu-id="a24da-120">Nella finestra di dialogo **Importa**, selezionare il documento che definisce il layout di report e scegliere **Apri**.</span><span class="sxs-lookup"><span data-stu-id="a24da-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>  
  
 <span data-ttu-id="a24da-121">Il layout del report personalizzato originale viene sostituito con il layout di report importato.</span><span class="sxs-lookup"><span data-stu-id="a24da-121">The original custom report layout is replaced with the imported report layout.</span></span>
