# <a name="design-details-inventory-valuation"></a><span data-ttu-id="4b6f7-103">Dettagli di progettazione: Valutazione di magazzino</span><span class="sxs-lookup"><span data-stu-id="4b6f7-103">Design Details: Inventory Valuation</span></span>
<span data-ttu-id="4b6f7-104">La valutazione magazzino è la determinazione del costo assegnato a un articolo di magazzino, espresso dalla seguente equazione.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-104">Inventory valuation is the determination of the cost that is assigned to an inventory item, as expressed by the following equation.</span></span>  

<span data-ttu-id="4b6f7-105">Magazzino finale = magazzino iniziale + acquisti netti – costo dei beni venduti</span><span class="sxs-lookup"><span data-stu-id="4b6f7-105">Ending inventory = beginning inventory + net purchases – cost of goods sold</span></span>  

<span data-ttu-id="4b6f7-106">Il calcolo della valutazione di magazzino utilizza il campo **Importo costo (effettivo)** dei movimenti di valorizzazione per l'articolo.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-106">The calculation of inventory valuation uses the **Cost Amount (Actual)** field of the value entries for the item.</span></span> <span data-ttu-id="4b6f7-107">I movimenti vengono classificati in base al tipo di movimento che corrisponde ai componenti di costo, al costo diretto, al costo indiretto, allo scostamento, alla rivalutazione e all'arrotondamento.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-107">The entries are classified according to the entry type that corresponds to the cost components, direct cost, indirect cost, variance, revaluation, and rounding.</span></span> <span data-ttu-id="4b6f7-108">Per ulteriori informazioni, vedere [Dettagli di progettazione: Componenti costo](design-details-cost-components.md).</span><span class="sxs-lookup"><span data-stu-id="4b6f7-108">For more information, see [Design Details: Cost Components](design-details-cost-components.md).</span></span>  

<span data-ttu-id="4b6f7-109">I movimenti collegati tra loro, tramite il collegamento fisso o in base all'ipotesi generale di costo-flusso definita dal metodo di costing.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-109">Entries are applied against each other, either by the fixed application or according to the general cost-flow assumption defined by the costing method.</span></span> <span data-ttu-id="4b6f7-110">Un movimento di riduzione di magazzino può essere collegato a più di un movimento di aumento con differenti date di registrazione e possibilmente con costi di acquisto diversi.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-110">One entry of inventory decrease can be applied to more than one increase entry with different posting dates and possibly different acquisition costs.</span></span> <span data-ttu-id="4b6f7-111">Per ulteriori informazioni, vedere [Dettagli di progettazione: Collegamento articoli](design-details-item-application.md).</span><span class="sxs-lookup"><span data-stu-id="4b6f7-111">For more information, see [Design Details: Item Application](design-details-item-application.md).</span></span> <span data-ttu-id="4b6f7-112">Di conseguenza, il calcolo del valore di magazzino in una specifica data è basato sulla somma dei movimenti di valorizzazione positivi e negativi.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-112">Therefore, calculation of the inventory value for a given date is based on summing up positive and negative value entries.</span></span>  

## <a name="inventory-valuation-report"></a><span data-ttu-id="4b6f7-113">Report Valutazione magazzino</span><span class="sxs-lookup"><span data-stu-id="4b6f7-113">Inventory Valuation report</span></span>  
<span data-ttu-id="4b6f7-114">Per calcolare il valore di magazzino nel report **Valutazione magazzino**, il report inizia calcolando il valore del magazzino dell'articolo alla data di inizio specificata.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-114">To calculate the inventory value in the **Inventory Valuation** report, the report begins by calculating the value of the item’s inventory at a given starting date.</span></span> <span data-ttu-id="4b6f7-115">Aggiunge quindi gli aumenti del valore di magazzino e sottrae le diminuzioni del valore di magazzino fino a una data finale specificata.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-115">It then adds the value of inventory increases and subtracts the value of inventory decreases up to a given ending date.</span></span> <span data-ttu-id="4b6f7-116">Il risultato finale è il valore di magazzino alla data di fine.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-116">The end result is the inventory value on the ending date.</span></span> <span data-ttu-id="4b6f7-117">Il report calcola questi valori sommando i valori nel campo **Importo costo (effettivo)** nei movimenti di valorizzazione, utilizzando le date di registrazione come filtri.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-117">The report calculates these values by summing the values in the **Cost Amount (Actual)** field in the value entries, using the posting dates as filters.</span></span>  

<span data-ttu-id="4b6f7-118">Nel report stampato vengono sempre visualizzati gli importi effettivi, vale a dire il costo dei movimenti che sono stati registrati come fatturati.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-118">The printed report always shows actual amounts, that is, the cost of entries that have been posted as invoiced.</span></span> <span data-ttu-id="4b6f7-119">Se viene selezionato il campo Includi costo previsto della Scheda dettaglio Opzioni, nel report verrà stampato anche il costo previsto di movimenti che sono stati registrati come ricevuti o spediti.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-119">The report will also print the expected cost of entries that have posted as received or shipped, if you select the Include Expected Cost field on the Options FastTab.</span></span>  

> [!IMPORTANT]  
>  <span data-ttu-id="4b6f7-120">I valori del report **Valutazione magazzino** sono riconciliati con il conto giacenza magazzino nella contabilità generale, ovvero i movimenti di valorizzazione in questione sono stati registrati nella contabilità generale.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-120">Values in the **Inventory Valuation** report is reconciled with the Inventory account in the general ledger, meaning the value entries in question have been posted to the general ledger.</span></span>  

> [!IMPORTANT]  
>  <span data-ttu-id="4b6f7-121">Gli importi nelle colonne **Valore** del report sono basati sulla data di registrazione delle transazioni per un articolo.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-121">Amounts in the **Value** columns of the report are based on the posting date of transactions for an item.</span></span>  

## <a name="inventory-valuation---wip-report"></a><span data-ttu-id="4b6f7-122">Report Valutazione magazzino - WIP</span><span class="sxs-lookup"><span data-stu-id="4b6f7-122">Inventory Valuation - WIP report</span></span>  
<span data-ttu-id="4b6f7-123">Un'azienda manifatturiera deve determinare il valore di tre tipi di magazzino:</span><span class="sxs-lookup"><span data-stu-id="4b6f7-123">A manufacturing company needs to determine the value of three types of inventory:</span></span>  

* <span data-ttu-id="4b6f7-124">Magazzino di materie prime</span><span class="sxs-lookup"><span data-stu-id="4b6f7-124">Raw Materials inventory</span></span>  
* <span data-ttu-id="4b6f7-125">Magazzino WIP</span><span class="sxs-lookup"><span data-stu-id="4b6f7-125">WIP inventory</span></span>  
* <span data-ttu-id="4b6f7-126">Magazzino di prodotti finiti</span><span class="sxs-lookup"><span data-stu-id="4b6f7-126">Finished Goods inventory</span></span>  

<span data-ttu-id="4b6f7-127">Il valore del magazzino WIP è determinato dalla seguente equazione:</span><span class="sxs-lookup"><span data-stu-id="4b6f7-127">The value of WIP inventory is determined by the following equation:</span></span>  

* <span data-ttu-id="4b6f7-128">Magazzino WIP finale = Magazzino WIP iniziale + costi di produzione – costi delle merci lavorate</span><span class="sxs-lookup"><span data-stu-id="4b6f7-128">Ending WIP inventory = Beginning WIP inventory + manufacturing costs – cost of goods manufactured</span></span>  

<span data-ttu-id="4b6f7-129">Come per il magazzino acquistato, i movimenti di valorizzazione forniscono la base della valutazione di magazzino.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-129">As for purchased inventory, the value entries provide the basis of the inventory valuation.</span></span> <span data-ttu-id="4b6f7-130">Il calcolo viene eseguito utilizzando i valori del campo **Importo costo (effettivo)** dei movimenti di valorizzazione di capacità e dell'articolo associati a un ordine di produzione.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-130">The calculation is made using the values in the **Cost Amount (Actual)** field of the item and capacity value entries associated with a production order.</span></span>  

<span data-ttu-id="4b6f7-131">Lo scopo della valutazione di magazzino WIP consiste nel determinare il valore degli articoli la cui produzione non è ancora stata completata alla data stabilita.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-131">The purpose of WIP inventory valuation is to determine the value of the items whose manufacturing has not yet been completed on a given date.</span></span> <span data-ttu-id="4b6f7-132">Di conseguenza il valore di magazzino WIP è basato sui movimenti di valorizzazione correlati ai movimenti contabili capacità e consumo.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-132">Therefore the WIP inventory value is based on the value entries related to the consumption and capacity ledger entries.</span></span> <span data-ttu-id="4b6f7-133">I movimenti contabili del consumo devono essere completamente fatturati alla data di valutazione.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-133">Consumption ledger entries must be completely invoiced at the date of the valuation.</span></span> <span data-ttu-id="4b6f7-134">Di conseguenza, il report **Valutazione magazzino - WIP** mostra i costi che rappresentano il valore di magazzino WIP in due categorie: consumo e capacità.</span><span class="sxs-lookup"><span data-stu-id="4b6f7-134">Therefore, the **Inventory Valuation – WIP** report shows the costs representing the WIP inventory value in two categories: consumption and capacity.</span></span>
