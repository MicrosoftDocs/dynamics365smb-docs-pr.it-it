# <a name="how-to-create-special-orders"></a><span data-ttu-id="17402-104">Procedura: Creare ordini speciali</span><span class="sxs-lookup"><span data-stu-id="17402-104">How to: Create Special Orders</span></span>
<span data-ttu-id="17402-105">È possibile creare un ordine speciale per uno specifico articolo non presente in stock che deve venire spedito a un determinato cliente.</span><span class="sxs-lookup"><span data-stu-id="17402-105">You can create a special order for a specific nonstock item to be shipped to a specific customer.</span></span> <span data-ttu-id="17402-106">L'articolo viene spedito dal fornitore abituale alla warehouse dell'azienda e a questo punto può essere spedito al cliente da solo oppure insieme ad altri articoli di un ordine diverso.</span><span class="sxs-lookup"><span data-stu-id="17402-106">Your vendor ships the item to your warehouse and you can then ship the item on to your customer either independently or together with other items on another order.</span></span>  

<span data-ttu-id="17402-107">Gli ordini speciali implicano il collegamento dell'ordine di acquisto all'ordine di vendita allo scopo di garantire che lo specifico articolo non presente in stock venga prelevato e consegnato al cliente.</span><span class="sxs-lookup"><span data-stu-id="17402-107">Special orders imply that the purchase and sales order are linked to ensure that the specific nonstock item is picked and delivered to the customer.</span></span>  

<span data-ttu-id="17402-108">Prima di poter utilizzare questa funzionalità è necessario impostare le schede per il cliente, il fornitore e l'articolo interessati dall'ordine.</span><span class="sxs-lookup"><span data-stu-id="17402-108">Before you can use this feature, you must first set up the customer, vendor, and item cards necessary for the order.</span></span>  

## <a name="to-create-a-special-order"></a><span data-ttu-id="17402-109">Per creare un ordine speciale</span><span class="sxs-lookup"><span data-stu-id="17402-109">To create a special order</span></span>  
1.  <span data-ttu-id="17402-110">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "icona Cerca pagina o report"), immettere **Ordine di vendita**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="17402-110">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Sales Order**, and then choose the related link.</span></span>  
2. <span data-ttu-id="17402-111">Scegliere l'azione **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="17402-111">Choose the **New** action.</span></span> <span data-ttu-id="17402-112">Creare e compilare un  ordine di vendita per l'articolo.</span><span class="sxs-lookup"><span data-stu-id="17402-112">Create and fill in a  sales order for the item.</span></span> <span data-ttu-id="17402-113">Per ulteriori informazioni, vedere [Procedura: Vendere prodotti](sales-how-sell-products.md).</span><span class="sxs-lookup"><span data-stu-id="17402-113">For more information, see [How to: Sell Products](sales-how-sell-products.md).</span></span>
3.  <span data-ttu-id="17402-114">Nella Scheda dettaglio **Righe** compilare la riga di vendita.</span><span class="sxs-lookup"><span data-stu-id="17402-114">On the **Lines** FastTab, fill in the sales line.</span></span> <span data-ttu-id="17402-115">Nel campo **Codice acquisto** selezionare un codice di acquisto per il quale il campo **Ordine speciale** sia selezionato.</span><span class="sxs-lookup"><span data-stu-id="17402-115">In the **Purchasing Code** field, select a purchasing code that has the **Special Order** field selected.</span></span>

    <span data-ttu-id="17402-116">È quindi necessario creare un ordine di acquisto da una richiesta di approvvigionamento.</span><span class="sxs-lookup"><span data-stu-id="17402-116">You must now create a purchase order from a requisition worksheet.</span></span>  
4. <span data-ttu-id="17402-117">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "icona Cerca pagina o report"), immettere **Richiesta di approvvigionamento**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="17402-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Requisition Worksheet**, and then choose the related link.</span></span>  
5. <span data-ttu-id="17402-118">Scegliere l'azione **Ordine speciale** e scegliere l'azione **Ottieni ordini vendite**.</span><span class="sxs-lookup"><span data-stu-id="17402-118">Choose the **Special Order** action, and then choose the **Get Sales Orders** action.</span></span>  
6.  <span data-ttu-id="17402-119">Nella finestra **Prendi ordini vendite** vengono visualizzati i risultati in cui **Nr. documento** è il numero dell'ordine di vendita.</span><span class="sxs-lookup"><span data-stu-id="17402-119">In the **Get Sales Orders** window, show results where the **Document No.** is the sales order number.</span></span> <span data-ttu-id="17402-120">Scegliere il pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="17402-120">Choose the **OK** button.</span></span> <span data-ttu-id="17402-121">Viene creata una riga di richiesta di approvvigionamento per l'articolo.</span><span class="sxs-lookup"><span data-stu-id="17402-121">A requisition worksheet line is created for the item.</span></span>  
7.  <span data-ttu-id="17402-122">Nella riga di richiesta di approvvigionamento selezionare **Nuovo** nel campo **Messaggio azione**.</span><span class="sxs-lookup"><span data-stu-id="17402-122">On the requisition worksheet line, in the **Action Message** field, select **New**.</span></span>  
8.  <span data-ttu-id="17402-123">Nella finestra **Richiesta di approvv.** scegliere l'azione **Esegui messaggi di azione**.</span><span class="sxs-lookup"><span data-stu-id="17402-123">In the **Req. Worksheet** window, choose the **Carry Out Action Message** action.</span></span> <span data-ttu-id="17402-124">Viene aperta la finestra **Approv. - Esegui mess. azioni**.</span><span class="sxs-lookup"><span data-stu-id="17402-124">The **Carry Out Action Msg. - Req.** window opens.</span></span> <span data-ttu-id="17402-125">Scegliere il pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="17402-125">Choose the **OK** button.</span></span>  

    <span data-ttu-id="17402-126">Verrà quindi visualizzato un messaggio con l'informazione che gli ordini di acquisto sono stati creati.</span><span class="sxs-lookup"><span data-stu-id="17402-126">A message appears telling you that the purchase orders have been created.</span></span> <span data-ttu-id="17402-127">Scegliere il pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="17402-127">Choost the **OK** button.</span></span>  

<span data-ttu-id="17402-128">Un ordine di acquisto creato come ordine speciale per un ordine di vendita viene rispettato dal sistema di pianificazione poiché pareggia la domanda e l'offerta.</span><span class="sxs-lookup"><span data-stu-id="17402-128">A purchase order created as a special order for a sales order is respected by the planning system as it balances demand and supply.</span></span> <span data-ttu-id="17402-129">Ossia, l'ordine di acquisto (approvvigionamento) resta collegato all'ordine di vendita (domanda), anche se l'ordine di acquisto può approvvigionare un'altra domanda precedente.</span><span class="sxs-lookup"><span data-stu-id="17402-129">That is, the purchase order (supply) remains linked to the sales order (demand), even if that purchase order could supply another earlier demand.</span></span> <span data-ttu-id="17402-130">Per ulteriori informazioni, vedere [Dettagli di programmazione: Metodi di riordino](design-details-reservation-order-tracking-and-action-messaging.md).</span><span class="sxs-lookup"><span data-stu-id="17402-130">For more information, see [Design Details: Reordering Policies](design-details-reservation-order-tracking-and-action-messaging.md).</span></span>  

> [!NOTE]  
>  <span data-ttu-id="17402-131">Se l'elemento è già prenotato, non è possibile utilizzare la funzionalità ordine speciale.</span><span class="sxs-lookup"><span data-stu-id="17402-131">You cannot use the special order functionality if the item is already reserved.</span></span> <span data-ttu-id="17402-132">Pertanto, per gli articoli venduti in ordini speciali, assicurarsi che il campo **Impegno** nella scheda articolo non sia impostato su **Sempre**.</span><span class="sxs-lookup"><span data-stu-id="17402-132">Therefore, for items that are sold on special orders, make sure the **Reserve** field on the item card is not set to **Always**.</span></span>
