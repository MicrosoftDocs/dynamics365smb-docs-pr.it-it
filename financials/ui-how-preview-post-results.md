<span data-ttu-id="19077-104">In ogni documento e registrazione che è possibile contabilizzare, è possibile scegliere il pulsante **Anteprima registrazione** per esaminare i differenti tipi di movimenti che saranno creati quando si registra il documento o le registrazioni.</span><span class="sxs-lookup"><span data-stu-id="19077-104">On every document and journal that can be posted, you can choose the **Preview Posting** button to review the different types of entries that will be created when you post the document or journal.</span></span>

## <a name="to-preview-gl-entries-that-will-result-from-posting-a-purchase-invoice"></a><span data-ttu-id="19077-105">Per visualizzare in anteprima i movimenti C/G che deriveranno dalla registrazione di una fattura di acquisto</span><span class="sxs-lookup"><span data-stu-id="19077-105">To preview G/L entries that will result from posting a purchase invoice</span></span>
1. <span data-ttu-id="19077-106">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "icona Cerca pagina o report"), immettere **Fatture di acquisto**, quindi scegliere il collegamento correlato.</span><span class="sxs-lookup"><span data-stu-id="19077-106">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Purchase Invoices**, and then choose the related link.</span></span>
2. <span data-ttu-id="19077-107">Creare una fattura di acquisto.</span><span class="sxs-lookup"><span data-stu-id="19077-107">Create a purchase invoice.</span></span> <span data-ttu-id="19077-108">Per ulteriori informazioni, vedere [Procedura: Registrare gli acquisti](purchasing-how-record-purchases.md).</span><span class="sxs-lookup"><span data-stu-id="19077-108">For more information, see [How to: Record Purchases](purchasing-how-record-purchases.md).</span></span>
3. <span data-ttu-id="19077-109">Scegliere **Anteprima registrazione**.</span><span class="sxs-lookup"><span data-stu-id="19077-109">Choose **Preview Posting**.</span></span>
4. <span data-ttu-id="19077-110">Nella finestra **Anteprima registrazione** selezionare **Movimento C/G**, quindi scegliere **Mostra movimenti correlati**.</span><span class="sxs-lookup"><span data-stu-id="19077-110">In the **Posting Preview** window, select **G/L Entry**, and then choose **Show Related Entries**.</span></span>  
   <span data-ttu-id="19077-111">Nella finestra **Anteprima movimenti C/G** viene mostrato quali movimenti saranno creati quando si registra una fattura di acquisto.</span><span class="sxs-lookup"><span data-stu-id="19077-111">The **G/L Entries Preview** window shows which entries will be created when you post the purchase invoice.</span></span>
