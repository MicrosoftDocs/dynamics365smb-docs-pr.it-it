* <span data-ttu-id="9223a-119">Automaticamente, utilizzando la funzione **Importa rendiconto bancario** per compilare le righe a seconda degli importi degli estratti conto bancari basati su un file assegnato dalla banca.</span><span class="sxs-lookup"><span data-stu-id="9223a-119">Automatically, by using the **Import Bank Statement** function to fill in the lines according to actual bank statements based on a file provided by the bank.</span></span>
* <span data-ttu-id="9223a-120">Manualmente, utilizzando la funzione **Suggerisci righe** per compilare le righe con i movimenti contabili per fatture con pagamenti scaduti.</span><span class="sxs-lookup"><span data-stu-id="9223a-120">Manually, by using the **Suggest Lines** function to fill in the lines with ledger entries for invoices that have outstanding payments.</span></span>
