##<a name="to-use-the-getaddressio-uk-postcodes-extension-when-you-enter-an-address"></a><span data-ttu-id="376bc-107">Per utilizzare l'estensione dei codici postali di GetAddress.io per il Regno Unito quando viene immesso un indirizzo</span><span class="sxs-lookup"><span data-stu-id="376bc-107">To use the GetAddress.io UK Postcodes extension when you enter an address</span></span>

    > [!NOTE]  
>   <span data-ttu-id="376bc-109">L'opzione **Cercare l'indirizzo dal codice postale** è disponibile solo se il campo **Codice paese** è vuoto o contiene **GB**.</span><span class="sxs-lookup"><span data-stu-id="376bc-109">The **Look up address from postcode** option is available only if the **Country/Region Code** field is either empty, or contains **GB**.</span></span>

> [!TIP]
