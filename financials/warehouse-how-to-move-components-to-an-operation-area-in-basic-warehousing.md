>

## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="bf7b9-123">Per spostare componenti in un'area di operazione nelle configurazioni di warehouse di base</span><span class="sxs-lookup"><span data-stu-id="bf7b9-123">To move components to an operation area in basic warehouse configurations</span></span>  
1.  <span data-ttu-id="bf7b9-124">Scegliere l'icona ![Cerca pagina o report](media/ui-search/search_small.png "Cerca pagina o report"), immettere **Movimento di magazzino**, quindi scegliere il collegamento pertinente.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-124">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Inventory Movement**, and choose the relevant link.</span></span>  
2.  <span data-ttu-id="bf7b9-125">Nella Scheda dettaglio **Generale** compilare il campo **Nr.**</span><span class="sxs-lookup"><span data-stu-id="bf7b9-125">On the **General** FastTab, fill in the **No.**</span></span> <span data-ttu-id="bf7b9-126">.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-126">field.</span></span> <span data-ttu-id="bf7b9-127">È possibile premere INVIO per selezionare la numerazione.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-127">You can press the Enter key  to select from the number series.</span></span>  
3.  <span data-ttu-id="bf7b9-128">Nel campo **Cod. ubicazione** immettere l'ubicazione in cui si verifica la movimentazione.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-128">In the **Location Code** field, enter the location where the movement occurs.</span></span>  
4.  <span data-ttu-id="bf7b9-129">Scegliere l'azione **Prendi documenti origine**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-129">Choose the **Get Source Documents** action.</span></span> <span data-ttu-id="bf7b9-130">In alternativa, compilare il campo **Documento origine**, quindi fare clic sul pulsante **AssistEdit** nel campo **Nr. origine**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-130">Alternatively, fill in the **Source Document** field, and then choose the **AssistEdit** button in the **Source No.** field.</span></span>  
5.  <span data-ttu-id="bf7b9-131">Nella finestra **Documenti origine** selezionare l'ordine di assemblaggio per cui si desidera spostare i componenti, quindi fare clic sul pulsante **OK**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-131">In the **Source Documents** window, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>  

    <span data-ttu-id="bf7b9-132">Per ogni componente necessario che può essere spostato, viene generata una riga Prendere e una riga Mettere nella finestra **Movimenti di magazzino**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-132">For each needed component that can be moved, one Take line and one Place line are generated in the **Inventory Movements** window.</span></span> <span data-ttu-id="bf7b9-133">Tutti i campi tranne il campo **Qtà da gestire** sono precompilati in base alle righe del documento di origine.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-133">All fields except the **Qty. to Handle** field are prefilled according to the source document lines.</span></span> <span data-ttu-id="bf7b9-134">Il campo **Qtà da gestire** è uguale a zero fino a quando non si immette la quantità effettivamente spostata.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-134">The **Qty. to Handle** field is set to zero until you enter the quantity that you have actually moved.</span></span>  

    <span data-ttu-id="bf7b9-135">È possibile modificare il codice collocazione in una riga Prendere, ma solo in base alla disponibilità.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-135">You can change the bin code on a Take line but only according to availability.</span></span> <span data-ttu-id="bf7b9-136">Scegliendo il pulsante **AssistEdit** nel campo **Cod. collocazione** in una riga Prendere, la finestra **Contenuto collocazioni** viene aperta e mostra solo le collocazioni in cui il componente è disponibile.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** window opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="bf7b9-137">Non è possibile modificare il codice collocazione in una riga Mettere.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="bf7b9-138">Solo il codice collocazione definito nella riga componente del documento di origine viene accettato.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="bf7b9-139">Ciò supporta il principio secondo cui il ruolo che richiede un componente, ovvero un addetto all'assemblaggio in questa procedura, sa dove deve essere inserito il componente.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="bf7b9-140">Se si desidera inserire i componenti in una collocazione diversa, è necessario innanzitutto modificare il codice collocazione specificato nella riga componenti, quindi ricreare le righe movimenti di magazzino.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>  
6.  <span data-ttu-id="bf7b9-141">Nel campo **Qtà da gestire** immettere la quantità parziale o completa effettivamente spostata.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-141">In the **Qty. to Handle** field, enter the full or partial quantity that you have actually moved.</span></span> <span data-ttu-id="bf7b9-142">Il valore nelle righe Prendere e Mettere deve essere identico.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-142">The value on the Take and the Place lines must be the same.</span></span> <span data-ttu-id="bf7b9-143">In caso contrario, non è possibile registrare la movimentazione.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-143">Otherwise, you cannot register the movement.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="bf7b9-144">Come in altre attività di warehouse, è possibile suddividere la riga Posizione selezionando l'azione **Azioni** e scegliendo l'azione **Dividi riga**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-144">As in other warehouse activities, you can split the Place line by selecting the **Actions** action and then choosing the **Split Line** action.</span></span> <span data-ttu-id="bf7b9-145">In questo caso, la somma delle due righe divise Mettere deve essere uguale alla quantità nella riga Prendere.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-145">In that case, the sum of the two split Place lines must equal the quantity on the Take line.</span></span>  

7.  <span data-ttu-id="bf7b9-146">Al momento di registrare le movimentazioni effettuate, scegliere l'azione **Registra movimentazioni inv.**.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-146">When you are ready to register the movements that you have performed, choose the **Register Invt. Movement** action.</span></span>  

    <span data-ttu-id="bf7b9-147">I movimenti warehouse vengono creati in modo da riflettere la presenza dei componenti nelle collocazioni specificate nelle righe ordine di assemblaggio.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="bf7b9-148">A differenza di quando si spostano componenti con un prelievo magazzino, il consumo non viene registrato quando si registra un movimento di magazzino.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="bf7b9-149">Tale passaggio deve essere eseguito separatamente registrando l'output e il consumo dell'ordine di assemblaggio.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="bf7b9-150">Per ulteriori informazioni, vedere Ordine di assemblaggio.</span><span class="sxs-lookup"><span data-stu-id="bf7b9-150">For more information, see Assembly Order.</span></span>
